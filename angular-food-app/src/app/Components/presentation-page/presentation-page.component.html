FOOD APP

Welcome to Food App

My angular and spring boot project that mimics a food application, my objective
in this project was to build my first functional web app, and put to the test
some of my knowledge in full-stack development, all the UI is built with Angular Material,
in this project you will find features like functional login pages with ,
registration and password reset, food search with autocomplete suggestions,
an user account interface with the user informations, a table of foods that each
user have registered, the possibility of adding a new food and editing it,
also removing foods, and a cart page that shows the foods added from the home page.
